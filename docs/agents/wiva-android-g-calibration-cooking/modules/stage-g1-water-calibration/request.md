# Запрос (G1 — калибровка воды)

Реализация калибровки воды в wiva-android по образцу wiva_electron: `WaterCalibrationService`, вкладка «Обслуживание» → «Калибровка воды», ключ `WATER_CALIBRATION` в JsonStore, мок 1 с / 20 мл/с, unit-тесты расчётов, DoD по чеклисту модуля.

Сессия: `wiva-android-g-calibration-cooking/modules/stage-g1-water-calibration`.
