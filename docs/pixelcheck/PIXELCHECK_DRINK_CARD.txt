Pixel-perfect: карточка напитка (DrinkCard)
==========================================

Эталон в коде (веб): wiva_electron/.../DrinkCard/DrinkCard.module.scss
Размеры карточки: 301.333 x 156 px, фото 160 x 156, паддинги правой колонки: 20px, слева 0.

Как проверить 1:1 с макетом / с Electron
----------------------------------------
1) Скрин макета: экспорт из Figma (или скрин окна Electron на том же разрешении, что киоск).
2) Скрин Android: adb exec-out screencap -p > docs/pixelcheck/app_emulator_latest.png
   (файл обновляй вручную перед сравнением).
3) В Pencil (или Figma): два слоя на одном артборде, верхний — Multiply/Normal с opacity 40–50%,
   выровняй по левому верхнему углу карточки — смещения шрифта и паддингов видны сразу.

Файл .pen для Pencil
--------------------
1) Запусти настольное приложение Pencil (чтобы Cursor подключился по MCP).
2) В чате Cursor: попроси агента создать docs/pixelcheck/wiva_drink_card_pixelcheck.pen через Pencil MCP
   (batch_design: артборд 301x156 + два плейсхолдера под скрин макета и скрин приложения, верхний слой ~50% opacity).

Если в Cursor ошибка MCP: «WebSocket not connected to app: cursor» — плагин Pencil в Cursor
не видит открытое приложение: обнови Cursor, перезапусти Pencil, в настройках MCP убедись, что
сервер pencil включён; иногда помогает открыть .pen из Cursor (файл в проекте) после старта Pencil.

Пока MCP недоступен — собери overlay вручную: два Image в одном фрейме.

Подложки для ручной сборки в Pencil
----------------------------------
- docs/pixelcheck/app_emulator_latest.png — последний снимок эмулятора (обновляется командой выше).

Почему «плыли» размеры на Android
---------------------------------
Типографика 20/26 из макета задаётся в sp; при fontScale != 1.0 высота текста меняется.
Колонка с названием теперь с Modifier.weight(1) + heightIn(min=0), чтобы заголовок не раздувал карточку.
